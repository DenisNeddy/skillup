Цели курса:

  Соединить базовые понятия JavaScript с полноценным использованием фреймворков.

  Облегчить переход от базового JavaScript к фреймворкам типа React, помочь понять концепции стейтов, пропсов, компонентов и их взаимодействия.

  Предоставить углубленные знания по объектно-ориентированному программированию (ООП), SOLID, Event Loop, таймерам, Document Object Model (DOM), модульности, npm, асинхронному программированию и обработке промисов.

  Подготовить к разработке сложных проектов на фронтенде и бэкенде.

Содержание курса:

  Основы для перехода к фреймворкам:

    Понимание компонентов, стейтов, пропсов и их связи с базовым JavaScript.

  Углубление в ООП и SOLID:

    Принципы проектирования приложений на основе ООП.

    Детальное изучение работы Event Loop и взаимосвязь промисов с Event Loop.

    Понимание DOM, работы с событиями, модульности и npm.

  Практическая часть:
    Разработка полноценного Single Page Application (SPA).

    Создание компонентной структуры с использованием сборщика проектов.

    Реализация мини-фреймворка для повторного использования компонентов и реактивности приложения.

Для кого этот курс:
  Новички в разработке, имеющие базовые знания JavaScript, HTML, и CSS.

    Опытные разработчики, желающие систематизировать и углубить свои знания в JavaScript и понять продвинутые концепции.

После курса:
  У вас будет прочная основа JavaScript для движения как в направлении бэкенда (Node.js, Nest, Fastify, Express и др.), так и веб-разработки, включая фреймворки (Angular, React, Vue, Svelte и др.), мобильные приложения (React Native, NativeScript) и тестирование (Cypress, Playwright).
